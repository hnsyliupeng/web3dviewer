web3dviewer

web3dviewer is flexible JavaScript component for efficient, convenient and easy presentation of 3D objects using modern web-browser technologies.
The source code is published under the MIT licence.

  • Demo 1 - Basic usage. Simple box mesh with inertia.
  • Demo 2 - Hinge model (9000 faces). Progressive, noinertia. Test your device performance.
  • Demo 3 - Two independent components. Hinge and arm models.
  • Demo 4 - Four independent components.
  • Demo 5 - One object, two connected components.
  • Demo 6 - Three views at once.
  • Demo 7 - Hinge model (9000 faces). No progressive. For streaming comparision.
